# Intro
## 강의 환경 구성
- 간단한 게임 제작 라이브러리 **pygame** 설치 필요
    - `$ python3 -m pip install pygame --user`
- 이미지 처리를 위한 **OpenCV** 설치 필요
    - `$ python3 -m pip install opencv-python --user`

## 알고리즘 테스트 환경
- 간단한 게임을 이용해서 강화학습의 성능 테스트를 할 예정
